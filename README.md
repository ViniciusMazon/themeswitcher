# 🛸 Tecnologias

* ReactJS
* Styled-components



# ⚙️ Como utilizar

Clone o repositório:

```bash
git clone https://github.com/ViniciusMazon/themeswitcher.git
```

Instale as dependências:

```bash
yarn
```
Rode utilizando o script

```bash
yarn start
```
